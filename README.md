# 1_ERT_Raw_Data
For this thesis, two electrical resistivity tomography (ERT) surveys have been conducted particularly in Wendelsheim and Pfäffingen, near to Tübingen, Germany. In this repository, I am uploading the raw ERT data collected from the field. Furthermore, I am also uploading separate data set for DD and MG, configurations used in this study, specifically their raw file. These all raw data was collected by Syscal pro instrument. For Wendelsheim, data were collected on June 25/ 2025 and for Pfäffingen, data was collected in two days: April (13,14)/ 2026.
