
# 1_ERT_Raw_Data

## Overview

This repository contains the raw Electrical Resistivity Tomography (ERT) data sets acquired during the field campaigns conducted for my Master's thesis in the Ammer Valley, near Tübingen, Germany.

Two survey sites are included:

- **Wendelsheim** (survey conducted on **25 June 2025**)
- **Pfäffingen** (survey conducted on **13–14 April 2026**)

All measurements were acquired using the **IRIS Instruments Syscal Pro** resistivity imaging system.

In addition to the complete raw datasets, separate datasets containing only the **Dipole–Dipole (DD)** and **Multi-Gradient (MG)** measurements are also provided.

---

## Repository Contents

| File | Description |
|------|-------------|
| **20250625_Wendelsheim_promys.xlsx** | Complete raw ERT dataset acquired at the Wendelsheim survey site on 25 June 2025. Contains all measurements recorded during the field campaign. |
| **202604_13_14_Pfaffingen_promys.xlsx** | Complete raw ERT dataset acquired at the Pfäffingen survey site on 13–14 April 2026. Contains all measurements recorded during the field campaign. |
| **dd_only_Wendelsheim.xlsx** | Dipole–Dipole (DD) measurements extracted from the Wendelsheim dataset. |
| **mg_only_Wendelsheim.xlsx** | Mid-Gradient (MG) measurements extracted from the Wendelsheim dataset. |
| **dd_only_Pfaffingen.xlsx** | Dipole–Dipole (DD) measurements extracted from the Pfäffingen dataset. |
| **mg_only_Pfaffingen.xlsx** | Mid-Gradient (MG) measurements extracted from the Pfäffingen dataset. |

---

## Instrumentation

- **Instrument:** IRIS Instruments Syscal Pro
- **Method:** Electrical Resistivity Tomography (ERT)

---

## Purpose

These datasets are provided to ensure the transparency and reproducibility of the analyses presented in the thesis. They can be used for:

- ERT preprocessing
- Data quality assessment
- Array-specific analysis (DD and MG)
- Inversion using software such as PyGIMLi, BERT, or ResIPy etc.
- Reproducibility of the published results
